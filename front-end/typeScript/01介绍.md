# 介绍
`TypeScript`是什么:
1. 以`JavaScript`为基础构建的语言
2. 一个`JavaScript`的超集
3. 可以在任何支持`JavaScript`的平台中执行
4. `TypeScript`扩展了`JavaScript`并添加类型
5. `TS`不能被`JS`解析器直接执行

`TypeScript`较`JS`增加的特性:
- 类型
- 支持ES的新特性
- 强大的开发工具
- 添加ES不具备的新特性
- 丰富的配置选项

# 搭建`TypeScript`开发环境
1. 下载`Node.js`
2. 安装`Node.js`
3. 使用npm全局安装`TypeScript`
4. 创建一个`ts`文件
5. 使用`tsc`对ts文件进行编译
   - 进入命令行
   - 进入ts文件所在目录
   - 执行命令： tsc xxx.ts
